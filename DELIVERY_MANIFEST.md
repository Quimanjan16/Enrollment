## 🔄 Data Flow Supported

### Payment Flow
```
Tuition ÷ 4 → 4 Equal Installments
↓
+Additional Fee → Auto-splits across unpaid schedules
↓
-Scholarship → Auto-deducted from unpaid schedules
↓
= Final Amount Due per installment
```

### Enrollment Flow
```
Student Profile → Registrar creates
↓
Documents Upload → Registrar verifies
↓
Enrollment Create → Cashier processes
↓
Payment Schedule → 4 installments auto-created
↓
Payment Recording → Cashier records
↓
Assessment Verify → Assessment confirms
↓
Exam Eligible → Student can take exam
```

### Scholarship Flow
```
Scholarship Application → Created by parent/student
↓
Assessment Review → Assessment examines
↓
Approval → Scholarship approved
↓
Auto Deduction → Amount deducted from unpaid payments
↓
All Users See Update → Real-time sync
```
